### Hello World

`app.go`
```go
package main

import (
	"fmt"
	"net/http"
	"time"
)

func Handler(w http.ResponseWriter, r *http.Request) {
	fmt.Println(time.Now())
}

func main() {
	http.HandleFunc("/", Handler)
	http.ListenAndServe(":8000", nil)
}
```

```bash
$ go run app.go
```

$ curl localhost:8000

	http.HandleFunc("/", Handler)
